# Thragg — Shared Rules

## Versioning

When making any change to this repo, bump the patch version (e.g. `1.2.0` → `1.2.1`) in both:
- `.claude-plugin/marketplace.json` — `metadata.version` and `plugins[0].version`
- `.claude-plugin/plugin.json` — `version`
